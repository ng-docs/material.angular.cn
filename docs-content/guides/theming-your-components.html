使用 Angular Material 的主题体系为你自己的组件设置主题

你可以将 Angular Material 基于 Sass 的主题体系用于自定义组件。

从主题中读取样式值

如主题指南中所述，主题是一个 Sass 映射表，其中包含用于自定义组件的样式值。 Angular Material 提供了一些用于从该数据结构中读取值的 API。

读取颜色值

要从主题中读取颜色值，可以使用 get-theme-color Sass 函数。此函数支持读取应用颜色调色板（主色调、强调色和警告色）以及前景和背景调色板的颜色。 get-theme-color 接受三个参数：要读取的主题、调色板的名称和颜色的名称。

每个颜色调色板（主色调、强调色和警告色）都支持读取以下命名颜色：

颜色名称 描述
default 此调色板的默认颜色
lighter 此调色板颜色的浅色版本
darker 此调色板颜色的深色版本
text 此调色板的文本颜色
default-contrast 与该调色板的默认颜色形成对比的颜色
lighter-contrast 与该调色板的浅色形成对比的颜色
darker-contrast 与该调色板的深色形成对比的颜色
[色调] 调色板的 [色调] 颜色。
[色调] 可以是以下之一：50、100、200、300、400、500、600、700、800、900、A100、A200、A400、A700
[色调]-对比 与调色板的 [色调] 颜色形成对比的颜色。
[色调] 可以是以下之一：50、100、200、300、400、500、600、700、800、900、A100、A200、A400、A700

背景调色板支持读取以下命名颜色：

颜色名称 描述
status-bar 状态栏的背景颜色
app-bar 应用栏的背景颜色
background 应用的背景颜色
hover 悬停覆盖的背景颜色
card 卡片的背景颜色
dialog 对话框的背景颜色
raised-button 凸起按钮的背景颜色
selected-button 选中按钮的背景颜色
selected-disabled-button 选中禁用按钮的背景颜色
disabled-button 禁用按钮的背景颜色
focused-button 获得焦点的按钮的背景颜色
disabled-button-toggle 禁用的按钮切换的背景颜色
unselected-chip 未选中的芯片的背景颜色
disabled-list-option 禁用的列表选项的背景颜色
tooltip 工具提示的背景颜色

前景色调板支持读取以下命名颜色：

颜色名称 描述
base 基础前景色，可用于颜色混合或创建自定义不透明度前景色
divider 分隔线的颜色
dividers (分隔线的别名)
disabled-text 禁用文本的颜色
disabled (禁用文本的别名)
disabled-button 禁用按钮文本的颜色
elevation 颜色升高阴影
hint-text 提示文本的颜色
secondary-text 次要文本的颜色
icon 图标的颜色
icons (图标的别名)
text 文本的颜色

除了读取特定颜色外，你还可以使用 get-theme-type Sass 函数来确定主题类型（亮色或暗色）。此函数接受一个参数，即主题，并返回 light 或 dark。

请查看以下从主题读取一些颜色的示例：

$theme: mat.define-dark-theme(...);

$primary-default: mat.get-theme-color($theme, primary, default);
$accent-a100: mat.get-theme-color($theme, accent, A100);
$warn-500-contrast: mat.get-theme-color($theme, warn, 500-contrast);
$foreground-text: mat.get-theme-color($theme, foreground, text);
$background-card: mat.get-theme-color($theme, background, card);
$type: mat.get-theme-type($theme);
$custom-background: if($type == dark, #030, #dfd);

读取排版值

要从主题读取排版值，可以使用 get-theme-typography Sass 函数。此函数支持从主题中定义的排版级别读取排版属性。有两种调用此函数的方法。

第一种调用方法是只传递主题和排版级别，以获取基于该级别设置的简写 font 属性。（注意： letter-spacing 无法在 font 简写中表达，因此必须单独应用）。

第二种调用方法是传递主题、排版级别以及你想要的特定字体属性： font-family、 font-size、 font-weight、 line-height 或 letter-spacing。

可用的排版级别为：

名称 描述
headline-1 一次性标题，通常位于页面顶部（例如英雄标题）。
headline-2 一次性标题，通常位于页面顶部（例如英雄标题）。
headline-3 一次性标题，通常位于页面顶部（例如英雄标题）。
headline-4 一次性标题，通常位于页面顶部（例如英雄标题）。
headline-5 对应于 <h1> 标签的节标题。
headline-6 对应于 <h2> 标签的节标题。
subtitle-1 对应于 <h3> 标签的节标题。
subtitle-2 对应于 <h4> 标签的节标题。
body-1 基本正文文本。
body-2 次要正文文本。
caption 较小的正文和提示文本。
button 按钮和锚点。

请查看以下从主题读取一些排版设置的示例：

$theme: mat.define-dark-theme(...);

body {
  font: mat.get-theme-typography($theme, body-1);
  letter-spacing: mat.get-theme-typography($theme, body-1, letter-spacing);
}

读取密度值

要从主题读取密度比例，可以使用 get-theme-density Sass 函数。此函数接受一个主题并返回密度比例（0、-1、-2、-3、-4 或 -5）。

请查看以下从主题读取密度比例的示例：

$theme: mat.define-dark-theme(...);

$density-scale: mat.get-theme-density($theme);

检查为主题配置了哪些维度

根据主题的创建方式，它可能没有为所有主题维度（基础、颜色、排版、密度）配置数据。你可以通过调用 theme-has Sass 函数来检查主题是否为特定维度配置了数据，并将主题和要检查的维度传递给它。

请查看以下检查主题的配置维度的示例：

$theme: mat.define-dark-theme(...);

$has-base: mat.theme-has($theme, base);
$has-color: mat.theme-has($theme, color);
$has-typography: mat.theme-has($theme, typography);
$has-density: mat.theme-has($theme, density);

分离主题样式

每个 Angular Material 组件都有一个 Sass 文件，该文件定义了用于自定义该组件的颜色和排版的 mixin。例如：MatButton 具有 button-color 和 button-typography 这两个 mixin。它们分别生成该组件的所有颜色和排版样式。

你可以通过定义自己的 mixin，在你的组件中模仿这种结构。这些 mixin 应该接受一个 Angular Material 主题，它们可以从中读取颜色和排版值。然后，你可以将这些 mixin 与 Angular Material 自己的 mixins 一起包含在你的应用程序中。

分步示例

为了说明如何融入 Angular Material 的主题体系，我们可以看一个自定义轮播组件的示例。 此轮播组件从单一文件 carousel.scss 开始，其中包含结构、颜色和排版样式。该文件包含在组件的 styleUrls 中。

// carousel.scss

.my-carousel {
  display: flex;
  font-family: serif;
}

.my-carousel-button {
  border-radius: 50%;
  color: blue;
}

步骤 1：将基于主题的样式提取到单独的文件中

为了更改此文件以融入 Angular Material 的主题体系，我们将这些样式拆分为两个文件，把颜色和排版样式移动到 mixin 中。按照惯例，新文件名以 -theme 结尾。此外，该文件以下划线 (_) 开头，以表明这是一个 Sass “部分（partial）文件”。关于“部分文件”的更多信息，请参阅 Sass 文档。

// carousel.scss

.my-carousel {
  display: flex;
}

.my-carousel-button {
  border-radius: 50%;
}

// _carousel-theme.scss

@mixin color($theme) {
  .my-carousel-button {
    color: blue;
  }
}

@mixin typography($theme) {
  .my-carousel {
    font-family: serif;
  }
}

第 2 步：使用主题中的值

现在，这些基于主题的样式都存在于 mixin 中，我们可以从传给这些 mixin 的主题中提取我们需要的值。

// _carousel-theme.scss

@use 'sass:map';
@use '@angular/material' as mat;

@mixin color($theme) {
  .my-carousel-button {
    // Read the 500 hue from the primary color palette.
    color: mat.get-theme-color($theme, primary, 500);
  }
}

@mixin typography($theme) {
  .my-carousel {
    // Get the headline font from the theme.
    font: mat.get-theme-typography($theme, headline-1);
  }
}

第 3 步：添加主题 mixin

为方便起见，我们可以添加一个包含颜色和排版的 theme mixin。如果指定了配置，则此主题 mixin 应仅生成每种颜色和排版的样式。

// _carousel-theme.scss

@use 'sass:map';
@use '@angular/material' as mat;

@mixin color($theme) {
  .my-carousel-button {
    // Read the 500 hue from the primary color palette.
    color: mat.get-theme-color($theme, primary, 500);
  }
}

@mixin typography($theme) {
  .my-carousel {
    // Get the headline font from the theme.
    font: mat.get-theme-typography($theme, headline-1);
  }
}

@mixin theme($theme) {
  @if mat.theme-has($theme, color) {
    @include color($theme);
  }

  @if mat.theme-has($theme, typography) {
    @include typography($theme);
  }
}

第 4 步：在你的应用程序中包含主题 mixin

现在你已经定义了轮播组件的主题 mixin，可以将此 mixin 与其他主题 mixin 一起包含在你的应用程序中。

@use '@angular/material' as mat;
@use './path/to/carousel-theme' as carousel;

@include mat.core();

$my-primary: mat.define-palette(mat.$indigo-palette, 500);
$my-accent: mat.define-palette(mat.$pink-palette, A200, A100, A400);

$my-theme: mat.define-light-theme((
 color: (
   primary: $my-primary,
   accent: $my-accent,
 ),
 typography: mat.define-typography-config(
    $font-family: serif,
  );
));

@include mat.all-component-themes($my-theme);
@include carousel.theme($my-theme);
