Angular CDK 平台 API 参考

import {PlatformModule} from '@angular/cdk/platform';

服务

Platform

本服务通过比较 userAgent 字符串和检查浏览器特定的全局属性来检测当前平台。

属性
名称 描述

ANDROID: boolean

当前的平台是不是 Android 版。

BLINK: boolean

当前的渲染引擎是否为 Blink。

EDGE: boolean

当前的浏览器是否为 Microsoft Edge。

FIREFOX: boolean

当前的浏览器是否为 Firefox。

IOS: boolean

目前的平台是否为 Apple iOS。

SAFARI: boolean

当前的浏览器是否为 Safari。

TRIDENT: boolean

当前的渲染引擎是否为 Microsoft Trident。

WEBKIT: boolean

当前的渲染引擎是否为 WebKit。

isBrowser: boolean

Angular 应用是否正在浏览器中渲染。

函数

getSupportedInputTypes
返回值
Set<string>

该浏览器支持的输入类型。
supportsPassiveEventListeners

检查用户的浏览器是否支持被动事件侦听器。参见：https://github.com/WICG/EventListenerOptions/blob/gh-pages/explainer.md
返回值
boolean

normalizePassiveListenerOptions

将 AddEventListener 对象规范化为可以在任何浏览器中传给 addEventListener 的，无论其是否支持 options 参数。
参数

options

AddEventListenerOptions

要规范化的对象。
返回值
AddEventListenerOptions | boolean

supportsScrollBehavior

检查浏览器是否支持滚动行为。
返回值
boolean

getRtlScrollAxisType

检查此浏览器使用的 RTL 滚动轴的类型。在撰写本文时，Chrome 浏览器为 NORMAL（正常），Firefox 和 Safari 为 NEGATED（否），而 IE＆Edge 为 INVERTED（反向）。
返回值
RtlScrollAxisType
