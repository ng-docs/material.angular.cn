接口

StepHarnessFilters

一组可用于过滤 MatStepHarness 实例列表的条件。

属性
名称 描述

completed: boolean

仅找到已完成的步骤。

invalid: boolean

仅找到有错误的步骤。

label: string | RegExp

只查找标签与指定值匹配的实例。

selected: boolean

仅找到具有给定选定状态的步骤。

StepperHarnessFilters

一组可用于过滤 MatStepperHarness 实例列表的条件。

属性
名称 描述

orientation: StepperOrientation

仅找到方向与给定值匹配的实例。

StepperButtonHarnessFilters

一组可用于过滤 MatStepperNextHarness 和 MatStepperPreviousHarness 实例列表的条件。

属性
名称 描述

text: string | RegExp

只查找其文本内容匹配指定值的实例。
