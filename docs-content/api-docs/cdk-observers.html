import {ObserversModule} from '@angular/cdk/observers';

服务

ContentObserver

一种可注入的服务，它允许监视元素内容的变化。

方法
observe

观察元素的内容变化。
参数

element

Element

要观察内容变化的元素。
返回值
Observable<MutationRecord[]>

observe

观察元素的内容变化。
参数

element

ElementRef<Element>

要观察内容变化的元素。
返回值
Observable<MutationRecord[]>

指令

CdkObserveContent

每当关联元素的内容发生变化时触发回调指令。

选择器： [cdkObserveContent]

Exported as: cdkObserveContent
属性
名称 描述
@Input()

debounce: number

用于在发出更改事件时防抖。
@Input({ alias: 'cdkObserveContentDisabled', transform: booleanAttribute })

disabled: boolean

对内容的观察是否已禁用。此选项可用于断开与底层 MutationObserver 的连接，直到需要它为止。
@Output('cdkObserveContent')

event: EventEmitter<MutationRecord[]>

针对元素内容中每个变化发出通知。
