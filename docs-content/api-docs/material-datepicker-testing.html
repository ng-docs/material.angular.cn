接口

DatepickerInputHarnessFilters

一组可用于过滤日期选择器输入实例列表的条件。

属性
名称 描述

placeholder: string | RegExp

根据输入的占位符文本进行过滤。

value: string | RegExp

根据输入值过滤。

DatepickerToggleHarnessFilters

一组可用于过滤日期选择器切换器实例列表的条件。

CalendarHarnessFilters

一组可用于过滤日历实例列表的条件。

CalendarCellHarnessFilters

一组可用于过滤日历单元格实例列表的条件。

属性
名称 描述

active: boolean

根据是否是使用键盘导航激活的单元格进行过滤

disabled: boolean

根据是否禁用了该单元进行过滤。

inComparisonRange: boolean

根据单元格是否在比较范围内进行过滤。

inPreviewRange: boolean

根据单元格是否在预览范围内进行过滤。

inRange: boolean

根据单元格是否在主范围内进行过滤。

selected: boolean

根据是否选择了单元格进行过滤。

text: string | RegExp

根据单元格的文本进行过滤。

today: boolean

根据单元格是否代表今天的日期进行过滤。
