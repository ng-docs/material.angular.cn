Angular Material 进度指示器的 API 参考

import {MatProgressSpinnerModule} from '@angular/material/progress-spinner';

指令

MatProgressSpinner

选择器： mat-progress-spinner mat-spinner

Exported as: matProgressSpinner
属性
名称 描述
@Input()

color: string

进度指示器的主题调色板颜色。
@Input({ transform: numberAttribute })

diameter: number

进度圈的直径（用于设置 svg 的宽度和高度）。
@Input()

mode: ProgressSpinnerMode

进度条的模式吧。

此输入属性必须是以下值之一：determinate、indeterminate、buffer 和 query，默认为 'determinate'。会镜像到 mode 属性。
@Input({ transform: numberAttribute })

strokeWidth: number

进度圈的线宽。
@Input({ transform: numberAttribute })

value: number

进度条的值。默认为零。镜像到 aria-valuenow。

接口

MatProgressSpinnerDefaultOptions

默认的 mat-progress-spinner 选项，可以改写它们。

属性
名称 描述

color: ThemePalette

进度圈的默认颜色。

diameter: number

进度圈的直径。

strokeWidth: number

进度圈的线宽。

类型别名

ProgressSpinnerMode

进度圈的可能模式。

type ProgressSpinnerMode = 'determinate' | 'indeterminate';

常量

MAT_PROGRESS_SPINNER_DEFAULT_OPTIONS

注入令牌，用于改写 mat-progress-spinner 的默认选项。

const MAT_PROGRESS_SPINNER_DEFAULT_OPTIONS: InjectionToken<MatProgressSpinnerDefaultOptions>;
