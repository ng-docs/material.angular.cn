SharedResizeObserver

允许使用共享的 ResizeObserver 集合来观察多个元素上的调整大小事件。 建议共享 ResizeObserver 实例以提高性能（参见 https://github.com/WICG/resize-observer/issues/59）。

此类不共享单个 ResizeObserver，而是为每种观察到的框类型（'content-box'、'border-box' 和 'device-pixel-content-box'）创建一个 ResizeObserver。这避免了以后对 observe 的调用（使用与先前调用不同的框类型）影响到先前调用分派的事件。
