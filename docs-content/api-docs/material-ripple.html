Angular Material 波纹的 API 参考

import {MatRippleModule} from '@angular/material/core';

指令

MatRipple

选择器： [mat-ripple] [matRipple]

Exported as: matRipple
属性
名称 描述
@Input('matRippleAnimation')

animation: RippleAnimationConfig

涟漪动画的配置。允许修改涟漪的进入和退出动画持续时间。如果使用 NoopAnimationsModule 则动画持续时间将被覆盖。
@Input('matRippleCentered')

centered: boolean

涟漪是否总是起源于宿主元素边界的中心，而不是起源于 click 事件的位置。
@Input('matRippleColor')

color: string

所有涟漪的自定义颜色。
@Input('matRippleDisabled')

disabled: boolean

点击事件是否不会触发涟漪。仍然可以使用 launch() 方法手动启动涟漪。
@Input('matRippleRadius')

radius: number

如果设置，则为完全扩展时前景涟漪的半径（以像素为单位）。如果未设置，则半径将是从涟漪中心到宿主元素边界矩形的最远角的距离。
@Input('matRippleTrigger')

trigger: HTMLElement

收到点击事件时触发涟漪的元素。默认为指令的宿主元素。
@Input('matRippleUnbounded')

unbounded: boolean

涟漪是否应在组件范围之外可见。
方法
fadeOutAll

淡出当前显示的涟漪元素的所有元素。
fadeOutAllNonPersistent

淡出当前显示的所有非持续性涟漪元素。
launch

使用指定的涟漪配置启动手动涟漪。
参数

config

RippleConfig

手动涟漪的配置。
返回值
RippleRef

在相对于视口的指定坐标处手动启动涟漪。

x

number

涟漪淡入时沿 Y 轴的坐标。此坐标相对于视口。

手动涟漪的可选涟漪配置。
返回值
RippleRef

类

RippleRef

对先前启动的涟漪元素的引用。

属性
名称 描述

config: RippleConfig

用于波纹的波纹配置。

element: HTMLElement

波纹 HTML 元素的引用。

state: RippleState

涟漪的当前状态。
方法
fadeOut

淡出涟漪元素。

接口

RippleGlobalOptions

matRipple 的可配置选项。

属性
名称 描述

animation: RippleAnimationConfig

涟漪的动画持续时间的配置。其两个阶段具有不同的涟漪持续时间。如果使用 NoopAnimationsModule 则动画持续时间将被覆盖。

disabled: boolean

是否应禁用涟漪。仍然可以使用 launch() 方法手动启动涟漪。因此，焦点指示器仍会出现。

namespace: string

用于涟漪加载器的命名空间，允许在同一页面上存在多个实例。

terminateOnPointerUp: boolean

释放鼠标或触摸后，涟漪是否应立即开始逐渐消失。默认情况下，涟漪将等待输入动画完成以及鼠标或触摸释放。

RippleAnimationConfig

描述涟漪动画配置的接口。有两个动画阶段，其涟漪的持续时间也不同。

属性
名称 描述

enterDuration: number

进场动画的持续时间（以毫秒为单位）（从接触点开始扩展）。

exitDuration: number

离场动画（淡出）的持续时间（以毫秒为单位）。

类型别名

RippleConfig 

type RippleConfig = {
    color?: string;
    centered?: boolean;
    radius?: number;
    persistent?: boolean;
    animation?: RippleAnimationConfig;
    terminateOnPointerUp?: boolean;
};

常量

MAT_RIPPLE_GLOBAL_OPTIONS

可以用于指定全局涟漪选项的注入令牌。

const MAT_RIPPLE_GLOBAL_OPTIONS: InjectionToken<RippleGlobalOptions>;

defaultRippleAnimationConfig

没有指定显式动画配置时的默认涟漪动画配置。

const defaultRippleAnimationConfig: { enterDuration: number; exitDuration: number; };
