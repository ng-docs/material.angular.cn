接口

MatTableHarnessColumnsText

从按列组织的表中提取的文本。

MatRowHarnessColumnsText

从按列组织的表行中提取的文本。

CellHarnessFilters

一组标准，可用于过滤单元测试工具实例的列表。

属性
名称 描述

columnName: string | RegExp

仅查找其列名与给定值匹配的实例。

text: string | RegExp

只查找其文本内容匹配指定值的实例。

RowHarnessFilters

一组标准，可用于过滤表行测试工具实例的列表。

TableHarnessFilters

一组标准，可用于过滤表格测试工具实例的列表。
