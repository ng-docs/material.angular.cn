Angular Material 底部菜单的 API 参考。

import {MatBottomSheetModule} from '@angular/material/bottom-sheet';

服务

MatBottomSheet

用于触发 Material Design 底部操作表的服务。

方法
dismiss

关掉当前可见的底部操作表。
参数

结果 ?

R

要传递给底部操作表实例的数据。
open

打开一个包含指定组件的底部操作表。
参数

component

ComponentType<T>

要装入底部操作表的组件类型。

配置 ?

MatBottomSheetConfig<D>

额外的配置选项。
返回值
MatBottomSheetRef<T, R>

指向新打开的底部操作表的引用。
open

打开一个包含指定模板的底部操作表。
参数

template

TemplateRef<T>

要用作底部操作表内容的 TemplateRef。

配置 ?

MatBottomSheetConfig<D>

额外的配置选项。
返回值
MatBottomSheetRef<T, R>

指向新打开的底部操作表的引用。

类

MatBottomSheetConfig

打开底部操作表时使用的配置。

属性
名称 描述

ariaLabel: string | null

指定给底部操作表元素的 Aria 标签。

ariaModal: boolean

这个底部操作表是否是模态的。用于设置 aria-modal 属性。

autoFocus: AutoFocusTarget | string | boolean

底部工作表打开时应聚焦到的位置。。

backdropClass: string

背景板的自定义类。

closeOnNavigation: boolean

当用户在历史记录中前进或后退时，底部操作表是否应该关闭。请注意，这通常不包括单击某些链接（除非用户正在使用 HashLocationStrategy ）。

data: D | null

注入到子组件中的数据。

direction: Direction

底部操作表的文本布局方向。

disableClose: boolean

用户是否可以使用 escape 或单击外部来关闭底部操作表。

hasBackdrop: boolean

底部操作表是否有背景板。

panelClass: string | string[]

要添加到底部操作表容器中的额外 CSS 类。

restoreFocus: boolean

关闭底部操作表时是否应该把焦点还给以前拥有焦点的元素。

scrollStrategy: ScrollStrategy

用于底部操作表的滚动策略。

viewContainerRef: ViewContainerRef

用于放置底部操作表浮层的容器。

MatBottomSheetRef

引用来自底部操作表服务的底部操作表。

属性
名称 描述

componentRef: ComponentRef<T> | null

打开到底部表单中的组件的 ComponentRef。当使用 TemplateRef 打开底部表单时，此值将为 null。

disableClose: boolean | undefined

是否允许用户关闭底部操作表。

instance: T

该组件的实例组成了底部操作表的内容。
方法
afterDismissed

获取一个可观察对象，当底部操作表已经完成并关闭时，它会收到通知。
返回值
Observable<R | undefined>

afterOpened

获取一个可观察对象，当底部操作表已经打开并出现时，它会收到通知。
返回值
Observable<void>

backdropClick

获取一个可观察对象，当点击浮层的背景板时，它会发出数据。
返回值
Observable<MouseEvent>

dismiss

关闭底部操作表。
参数

结果 ?

R

要传递回底部操作表的数据。
keydownEvents

获取一个可观察对象，当指定浮层收到 keydown 事件时，它会发出数据。
返回值
Observable<KeyboardEvent>

类型别名

AutoFocusTarget

用于在对话框打开时自动将焦点设置到何处的选项

type AutoFocusTarget = 'dialog' | 'first-tabbable' | 'first-heading';

常量

MAT_BOTTOM_SHEET_DEFAULT_OPTIONS

这个注入令牌可以用来指定底部操作表的默认选项。

const MAT_BOTTOM_SHEET_DEFAULT_OPTIONS: InjectionToken<MatBottomSheetConfig<any>>;

MAT_BOTTOM_SHEET_DATA

这个注入令牌可以用来访问那些传入底部操作表的数据。

const MAT_BOTTOM_SHEET_DATA: InjectionToken<any>;

matBottomSheetAnimations

Material 底部操作表使用的动画。

const matBottomSheetAnimations: { readonly bottomSheetState: AnimationTriggerMetadata; };
