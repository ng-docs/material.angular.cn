指令

CdkAutofill

一个指令，用于监控输入的自动填充状态。

选择器： [cdkAutofill]

属性
名称 描述
@Output()

cdkAutofill: EventEmitter<AutofillEvent>

元素的自动填充状态发生变化时触发。

CdkTextareaAutosize

用于自动调整 textarea 大小以适应其内容的指令。

选择器： textarea[cdkTextareaAutosize]

Exported as: cdkTextareaAutosize
属性
名称 描述
@Input({ alias: 'cdkTextareaAutosize', transform: booleanAttribute })

enabled: boolean

是否启用了自动调整大小的功能
@Input('cdkAutosizeMaxRows')

maxRows: number

textarea 中的最大行数。
@Input('cdkAutosizeMinRows')

minRows: number

textarea 中的最小行数。
@Input()

placeholder: string

方法
reset

把 textarea 重置为原始大小
resizeToFitContent

调整 textarea 的大小以适应它的内容。
参数

force

boolean = false

是否强行重新计算高度。默认情况下，只有当自上一次调用后的值发生变化时，才会重新计算高度。

类型别名

AutofillEvent

当输入框的自动填充状态发生变化时发出的事件。

type AutofillEvent = {
    target: Element;
    isAutofilled: boolean;
};
