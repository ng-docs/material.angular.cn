接口

FormFieldHarnessFilters

一组可用于过滤 MatFormFieldHarness 实例列表的条件。

属性
名称 描述

floatingLabelText: string | RegExp

根据表单字段的浮动标签的文本进行过滤。

hasErrors: boolean

根据表单字段是否包含错误消息进行过滤。

isValid: boolean

根据表单字段值是否有效进行过滤。

ErrorHarnessFilters

一组可用于过滤错误组件测试工具实例列表的条件。

属性
名称 描述

text: string | RegExp

只查找其文本内容匹配指定值的实例。

类型别名

FormFieldControlHarness

可以绑定到表单字段的可能的控件测试工具。

type FormFieldControlHarness = MatInputHarness | MatSelectHarness | MatDatepickerInputHarness | MatDateRangeInputHarness;
