Angular Material 按钮的 API 参考。

import {MatButtonModule} from '@angular/material/button';

指令

MatButton

Material Design 按钮组件。用户与按钮交互以执行操作。 请参阅 https://material.io/components/buttons

MatButton 类应用于原生按钮元素，并根据 Material Design 规范来表现“文本（text）按钮”、“轮廓（outlined）按钮”和“实心（contained）按钮”的外观。 MatButton 还表现了一个额外的“平面按钮”外观，它匹配“实心按钮”但没有纵深。

选择器： button[mat-button] button[mat-raised-button] button[mat-flat-button] button[mat-stroked-button]

Exported as: matButton
属性
名称 描述
@Input({ transform: booleanAttribute, alias: 'aria-disabled' })

ariaDisabled: boolean | undefined

按钮的 aria-disabled 值。
@Input()

color: string | null

按钮的主题颜色调色板。
@Input({ transform: booleanAttribute })

disableRipple: boolean

是否已禁用涟漪效果。
@Input({ transform: booleanAttribute })

disabled: boolean

该按钮是否已禁用。
@Input({ transform: booleanAttribute })

disabledInteractive: boolean

本机禁用的按钮会阻止焦点和任何指针事件到达按钮。 在某些情况下，这可能不理想，因为它会阻止用户找出 按钮被禁用的原因（例如，通过工具提示）。

启用此输入将更改按钮，使其被设置为禁用样式，并将被 标记为 aria-disabled，但它将允许按钮接收事件和焦点。

请注意，通过启用此功能，你需要自己设置 tabindex，如果按钮不 应该可被 Tab 键访问，并且你必须阻止按钮操作（例如，表单提交）。

MatAnchor

用于锚元素的 Material Design 按钮组件。锚元素用于为用户提供链接，以便他们在不同的路由或页面之间导航。 请参阅 https://material.io/components/buttons
