Angular CDK 步进器 API 参考

import {CdkStepperModule} from '@angular/cdk/stepper';

指令

CdkStep

选择器： cdk-step

Exported as: cdkStep
属性
名称 描述
@Input('aria-label')

ariaLabel: string

选项卡的 Aria 标签。
@Input('aria-labelledby')

ariaLabelledby: string

到用来标注此选项卡的元素的引用。如果同时设置了 aria-label，则被清除。
@Input({ transform: booleanAttribute })

completed: boolean

步骤是否标记为已完成。
@Input({ transform: booleanAttribute })

editable: boolean

一旦标记为已完成，用户是否可以返回此步骤。
@Input()

errorMessage: string

当出现错误时显示的错误信息。
@Input({ transform: booleanAttribute })

hasError: boolean

步骤是否有错误。
@Input()

label: string

该步骤的纯文本标签。
@Input({ transform: booleanAttribute })

optional: boolean

是否可以不必完成此步骤。
@Input()

state: StepState

步骤的状态。
@Input()

stepControl: AbstractControlLike

该步骤的顶级抽象控件。
@Output('interacted')

interactedStream: EventEmitter<CdkStep>

当用户尝试离开此步骤时，将发出此信号。

content: TemplateRef<any>

步骤内容的模板。

interacted: false

用户是否试图从这一步离开。

stepLabel: CdkStepLabel

步进器标签的模板（如果存在）。
方法
reset

把步骤重置为初始状态。请注意，这也包括重置表单数据。
select

选择此步骤组件。

CdkStepper

选择器： [cdkStepper]

Exported as: cdkStepper
属性
名称 描述
@Input({ transform: booleanAttribute })

linear: boolean

是否检查前一个步骤的有效性。
@Input()

orientation: StepperOrientation

此步进器的方向。
@Input()

selected: CdkStep | undefined

选定的步骤。
@Input({ transform: numberAttribute })

selectedIndex: number

选定步骤的索引。
@Output()

selectedIndexChange: EventEmitter<number>

用于支持 [(selectedIndex)] 双向绑定的输出
@Output()

selectionChange: EventEmitter<StepperSelectionEvent>

选定的步骤发生变化时发出的事件。

steps: QueryList<CdkStep>

属于当前步进器的步骤（不包括那些来自嵌套步进器的步骤）。
方法
next

选择并让列表中的下一步获得焦点。
previous

选择并让列表中的上一步获得焦点。
reset

步进器的初始状态是 1。请注意，这包括清除表单数据。

CdkStepLabel

选择器： [cdkStepLabel]

CdkStepperNext

用来移动到步进器工作流下一步的按钮。

选择器： button[cdkStepperNext]

属性
名称 描述
@Input()

type: string

下一个按钮的类型如果没有指定，默认为 “submit”。

CdkStepperPrevious

用来移动到步进器工作流中上一步的按钮。

选择器： button[cdkStepperPrevious]

属性
名称 描述
@Input()

type: string

上一个按钮的类型如果没有指定，默认为 “button”。

CdkStepHeader

选择器： [cdkStepHeader]

方法
focus

让步骤头获得焦点。

类

StepperSelectionEvent

在选定值变化时发出的变更事件。

属性
名称 描述

previouslySelectedIndex: number

前一个步骤的索引。

previouslySelectedStep: CdkStep

以前选定的步骤实例。

selectedIndex: number

当前步骤的索引。

selectedStep: CdkStep

此步骤的实例已被选定。

接口

StepperOptions

步进器的可配置选项。

属性
名称 描述

displayDefaultIndicatorType: boolean

步进器是否应该显示默认的指示器类型。假设默认行为是 true。

showError: boolean

步进器是否应该显示错误状态。假定默认行为是 false。

类型别名

StepContentPositionState

步进器中每个步骤内容的位置状态，用于在选定的步骤改变时将内容转移到正确的位置。

type StepContentPositionState = 'previous' | 'current' | 'next';

StepperOrientation

步进器可能的方向。

type StepperOrientation = 'horizontal' | 'vertical';

StepState

每一步的状态。

type StepState = 'number' | 'edit' | 'done' | 'error' | string;

常量

STEP_STATE

枚举表示步骤的不同状态。

const STEP_STATE: { NUMBER: string; EDIT: string; DONE: string; ERROR: string; };

STEPPER_GLOBAL_OPTIONS

这个注入令牌可以用来指定全局的步进器选项。

const STEPPER_GLOBAL_OPTIONS: InjectionToken<StepperOptions>;
