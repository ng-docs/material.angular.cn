Angular Material 表单域 API 参考

import {MatFormFieldModule} from '@angular/material/form-field';

指令

MatLabel

mat-form-field 的浮动标签。

选择器： mat-label

MatError

要在表单字段下方显示的单个错误消息。

选择器： mat-error [matError]

属性
名称 描述
@Input()

id: string

MatHint

提示文本显示在表单字段控件的下方。

选择器： mat-hint

属性
名称 描述
@Input()

align: 'start' | 'end'

把提示标签对齐到行的开头还是结尾。
@Input()

id: string

提示的唯一 ID。用于表单字段控件中的 aria-describedby。

MatPrefix

要放在表单字段前面的前缀。

选择器： [matPrefix] [matIconPrefix] [matTextPrefix]

MatSuffix

后缀放在表单字段的末尾。

选择器： [matSuffix] [matIconSuffix] [matTextSuffix]

MatFormField

表单控件的容器，用来应用 Material Design 的样式和行为。

选择器： mat-form-field

Exported as: matFormField
属性
名称 描述
@Input()

appearance: MatFormFieldAppearance

表单字段的外观样式。
@Input()

color: ThemePalette

此表单字段的调色板。
@Input()

floatLabel: FloatLabelType

浮动标签是否应该始终是浮动的。
@Input()

hideRequiredMarker: boolean

是否应隐藏必填项标记。
@Input()

hintLabel: string

表单字段提示的文本。
@Input()

subscriptSizing: SubscriptSizing

表单字段是否应该为单行提示/错误文本保留空间（默认）或根据提示/错误内容的大小根据需要从 0px 增加间距。请注意，使用动态调整大小时，布局会在提示/错误文本更改时发生变化。
方法
getConnectedOverlayOrigin

获取一个 ElementRef 元素，它为附着到表单字段上的浮层提供相对于该元素定位。
返回值
ElementRef

getLabelId

获取 label 元素的 id。如果没有 label，则返回 null。
返回值
string | null

类

MatFormFieldControl

本接口允许控件工作在 MatFormField 中。

属性
名称 描述

autofilled: boolean

本输入框当前是否处于自动填充状态。如果控件上没有此属性，则认为它是 false。

controlType: string

控件类型的可选名称，可以根据控件类型来区分 mat-form-field 元素。表单字段会在其根元素中添加形如 mat-form-field-type-{{controlType}} 的类。

disabled: boolean

该控件是否已禁用。

empty: boolean

控件是否为空。

errorState: boolean

控件是否处于错误状态。

focused: boolean

控件是否有焦点。

id: string

这个控件的元素 ID。

ngControl: NgControl | AbstractControlDirective | null

获取这个控件的 AbstractControlDirective。

placeholder: string

该控件的占位符。

required: boolean

控件是否必填的。

shouldLabelFloat: boolean

MatFormField 标签是否应该尝试浮动。

stateChanges: Observable<void>

每当控件的状态发生变化时就会发出的流，这样一来父 MatFormField 就要运行变更检测。

userAriaDescribedBy: string

aria-describedby 的值应该和由 form-field 所设置的各个 described-by id 合并。

value: T | null

控件的值。
方法
onContainerClick

处理此控件的容器上的点击事件。
参数

event

MouseEvent

setDescribedByIds

设置当前描述该控件的元素 ID 列表。
参数

ids

string[]

接口

MatFormFieldDefaultOptions

表示可通过 MAT_FORM_FIELD_DEFAULT_OPTIONS 令牌配置的表单字段的默认选项。

属性
名称 描述

appearance: MatFormFieldAppearance

默认表单字段外观样式。

color: ThemePalette

表单字段的默认颜色。

floatLabel: FloatLabelType

表单字段的标签默认应该是 always、never 还是 auto（只在必要时）。

hideRequiredMarker: boolean

默认情况下是否应隐藏所需的标记。

subscriptSizing: SubscriptSizing

此表单字段是否要默认预留一行空间。

类型别名

FloatLabelType

可用的 floatLabel 值的类型。

type FloatLabelType = 'always' | 'auto';

MatFormFieldAppearance

表单字段可能的外观样式。

type MatFormFieldAppearance = 'fill' | 'outline';

SubscriptSizing

决定如何设置下标高度的行为。

type SubscriptSizing = 'fixed' | 'dynamic';

常量

MAT_ERROR

这个注入令牌可以用来引用 MatError 实例。它可以作为实际 MatError 类的备用令牌，直接使用实际类可能导致该类及其元数据无法被优化掉。

const MAT_ERROR: InjectionToken<MatError>;

MAT_PREFIX

这个注入令牌可以用来引用 MatPrefix 实例。它可以作为实际 MatPrefix 类的备用令牌，直接使用实际类可能导致该类及其元数据无法被优化掉。

const MAT_PREFIX: InjectionToken<MatPrefix>;

MAT_SUFFIX

这个注入令牌可以用来引用 MatSuffix 实例。它可以作为实际 MatSuffix 类的备用令牌，直接使用实际类可能导致该类及其元数据无法被优化掉。

const MAT_SUFFIX: InjectionToken<MatSuffix>;

MAT_FORM_FIELD

这个注入令牌可以用来注入一个 MatFormField 的实例。它可以作为实际 MatFormField 类的备用令牌，使用实际类会导致 MatFormField 类及其组件元数据无法优化掉。

const MAT_FORM_FIELD: InjectionToken<MatFormField>;

MAT_FORM_FIELD_DEFAULT_OPTIONS

注入令牌，可以用来配置应用中所有表单字段的默认选项。

const MAT_FORM_FIELD_DEFAULT_OPTIONS: InjectionToken<MatFormFieldDefaultOptions>;
