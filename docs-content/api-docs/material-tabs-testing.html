接口

TabHarnessFilters

一组可用于过滤 MatTabHarness 实例列表的条件。

属性
名称 描述

label: string | RegExp

只查找标签与指定值匹配的实例。

selected: boolean

仅查找所选状态与给定值匹配的实例。

TabGroupHarnessFilters

一组可用于过滤 MatTabGroupHarness 实例列表的条件。

属性
名称 描述

selectedTabLabel: string | RegExp

仅查找其选定选项卡标签与给定值匹配的实例。

属性
