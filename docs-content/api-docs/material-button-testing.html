参数

options

用来收窄搜索范围的选项：

  • selector 找到一个按钮，其宿主元素与给定的选择器匹配。
  • text 查找具有特定文本内容的按钮。
  • variant 查找与特定变体匹配的按钮。
返回值
HarnessPredicate<T>

使用给定选项配置过的 HarnessPredicate。

接口

ButtonHarnessFilters

一组可以用来过滤 MatButtonHarness 实例列表的条件。

属性
名称 描述

disabled: boolean

仅查找与给定禁用状态匹配的实例。

text: string | RegExp

只查找其文本内容匹配指定值的实例。

variant: ButtonVariant

仅查找具有变体的实例。
