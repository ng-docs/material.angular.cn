Angular Material 滑动条的 API 参考

import {MatSliderModule} from '@angular/material/slider';

指令

MatSlider

允许用户通过移动滑杆来从一系列值中进行选择。它的行为类似于原生的 <input type="range"> 元素。

选择器： mat-slider

Exported as: matSlider
属性
名称 描述
@Input()

color: ThemePalette

滑动条的调色板颜色。
@Input({ transform: booleanAttribute })

disableRipple: boolean

是否禁用滑动条中的涟漪效果。
@Input({ transform: booleanAttribute })

disabled: boolean

此滑块是否已禁用。
@Input({ transform: booleanAttribute })

discrete: boolean

此滑杆是否要在按下滑块时显示数值标签。
@Input()

displayWith: (value: number) => string

在滑块的标签上显示值之前用来格式化该值的函数。可以用来格式化非常大的数字，以便让它们适配滑杆。
@Input({ transform: numberAttribute })

max: number

滑杆所具有的最大值。
@Input({ transform: numberAttribute })

min: number

滑杆所具有的最小值。
@Input({ transform: booleanAttribute })

showTickMarks: boolean

此滑杆是否要沿滑杆轨道显示刻度线。
@Input({ transform: numberAttribute })

step: number

滑块处的值。

MatSliderThumb

将特定于滑杆的行为添加到 <mat-slider> 内的输入元素上的指令。 <mat-slider> 中最多可以放置两个。

如果使用一个，则必须使用选择器 matSliderThumb ，结果将是一个普通滑杆。如果使用两个，则必须使用选择器 matSliderStartThumb 和 matSliderEndThumb ，结果将是一个带有两个滑杆的滑块的范围滑杆。

选择器： input[matSliderThumb]

Exported as: matSliderThumb
属性
名称 描述
@Input({ transform: numberAttribute })

value: number

@Output()

dragEnd: EventEmitter<MatSliderDragEvent>

当停止拖动此滑杆的滑块时发出的事件。
@Output()

dragStart: EventEmitter<MatSliderDragEvent>

当开始拖动此滑杆的滑块时发出的事件。
@Output()

valueChange: EventEmitter<number>

value 更改时发出的事件。

percentage: number

与滑杆值一致的百分比值。

step: number
