Angular CDK 手风琴 API 参考

import {CdkAccordionModule} from '@angular/cdk/accordion';

指令

CdkAccordionItem

一个基本指令，供使用者扩展并改成组件装饰器。设置所有要由 CdkAccordion 的父指令管理的事件和属性。

选择器： cdk-accordion-item [cdkAccordionItem]

Exported as: cdkAccordionItem
属性
名称 描述
@Input({ transform: booleanAttribute })

disabled: boolean

此条目是否已禁用了。
@Input({ transform: booleanAttribute })

expanded: boolean

此条目是否已经展开了。
@Output()

closed: EventEmitter<void>

每次关闭此条目时都会发出本事件。
@Output()

destroyed: EventEmitter<void>

当此条目被销毁时会发出本事件。
@Output()

opened: EventEmitter<void>

每次打开此条目时都会发出本事件。

accordion: CdkAccordion

id: string

AccordionItem 的唯一 id。
方法
close

把此条目的展开状态设置为 false。
open

把此条目的展开状态设置为 true。
toggle

切换此条目的展开状态。

CdkAccordion

该指令的用途是管理 CdkAccordionItem 子组件的展开状态。

选择器： cdk-accordion [cdkAccordion]

Exported as: cdkAccordion
属性
名称 描述
@Input({ transform: booleanAttribute })

multi: boolean

手风琴是否应该允许同时展开多个手风琴条目。

id: string

一个只读的 id 值，用于在单选模式下进行协调。
方法
closeAll

允许多选时，可以同时关闭手风琴中的所有可用条目。
openAll

允许多选时，可以同时打开手风琴中的所有可用条目。

常量

CDK_ACCORDION

可用于引用 CdkAccordion 实例的注入令牌。它作为实际 CdkAccordion 类的替代令牌，可以避免不必要的类及其指令元数据的保留。

const CDK_ACCORDION: InjectionToken<CdkAccordion>;
