Angular Material 步进器的 API 参考

import {MatStepperModule} from '@angular/material/stepper';

服务

MatStepperIntl

国际化所需的步进器数据。

属性
名称 描述

changes: Subject<void>

只要这里的标签发生了变化就会发出流。如果标签在初始化后发生了变化，用它来通知组件。

completedLabel: string

用于向屏幕阅读器指示步骤已完成的标签。

editableLabel: string

用于指示步骤对屏幕阅读器是可编辑的标签。

optionalLabel: string

在可选步骤下要渲染的标签。

指令

MatStepLabel 继承自 CdkStepLabel

选择器： [matStepLabel]

MatStep 继承自 CdkStep

选择器： mat-step

Exported as: matStep
属性
名称 描述
@Input('aria-label')

ariaLabel: string

选项卡的 Aria 标签。
@Input('aria-labelledby')

ariaLabelledby: string

到用来标注此选项卡的元素的引用。如果同时设置了 aria-label，则被清除。
@Input()

color: ThemePalette

特定步骤的主题颜色。
@Input({ transform: booleanAttribute })

completed: boolean

步骤是否标记为已完成。
@Input({ transform: booleanAttribute })

editable: boolean

一旦标记为已完成，用户是否可以返回此步骤。
@Input()

errorMessage: string

当出现错误时显示的错误信息。
@Input({ transform: booleanAttribute })

hasError: boolean

步骤是否有错误。
@Input()

label: string

该步骤的纯文本标签。
@Input({ transform: booleanAttribute })

optional: boolean

是否可以不必完成此步骤。
@Input()

state: StepState

步骤的状态。
@Input()

stepControl: AbstractControlLike

该步骤的顶级抽象控件。
@Output('interacted')

interactedStream: EventEmitter<CdkStep>

当用户尝试离开此步骤时，将发出此信号。

content: TemplateRef<any>

步骤内容的模板。

interacted: false

用户是否试图从这一步离开。

stepLabel: MatStepLabel

步骤标签的内容，由 <ng-template matStepLabel> 提供。
方法
isErrorState

自定义错误状态匹配器，它还要检查交互式表单的有效性。
参数

control

AbstractControl<any, any>

form

NgForm | FormGroupDirective

返回值
boolean

reset

把步骤重置为初始状态。请注意，这也包括重置表单数据。
select

选择此步骤组件。

MatStepper 继承自 CdkStepper

选择器： mat-stepper mat-vertical-stepper mat-horizontal-stepper [matStepper]

Exported as: matStepper, matVerticalStepper, matHorizontalStepper
属性
名称 描述
@Input()

animationDuration: string

动画的持续时间。如果没有设置单位，将标准化为毫秒。
@Input()

color: ThemePalette

步进器中所有步骤的主题颜色。
@Input()

disableRipple: boolean

是否应为步骤头禁用涟漪。
@Input()

headerPosition: 'top' | 'bottom'

步进器标头的位置。仅适用于 horizontal 方向。
@Input()

labelPosition: 'bottom' | 'end'

标签应显示在底部还是尾部位置。仅适用于 horizontal 方向。
@Input({ transform: booleanAttribute })

linear: boolean

是否检查前一个步骤的有效性。
@Input()

orientation: StepperOrientation

此步进器的方向。
@Input()

selected: CdkStep | undefined

选定的步骤。
@Input({ transform: numberAttribute })

selectedIndex: number

选定步骤的索引。
@Output()

animationDone: EventEmitter<void>

转换到当前步骤完毕后发出的事件。
@Output()

selectedIndexChange: EventEmitter<number>

用于支持 [(selectedIndex)] 双向绑定的输出
@Output()

selectionChange: EventEmitter<StepperSelectionEvent>

选定的步骤发生变化时发出的事件。

steps: QueryList<MatStep>

属于当前步进器的步骤（不包括那些来自嵌套步进器的步骤）。
方法
next

选择并让列表中的下一步获得焦点。
previous

选择并让列表中的上一步获得焦点。
reset

步进器的初始状态是 1。请注意，这包括清除表单数据。
