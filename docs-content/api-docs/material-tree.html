MatTreeNodeDef extends CdkTreeNodeDef

具有 Material Design 样式的 CdkTree 节点定义的包装器。 保存该节点的模板和一个 when 谓词，用于描述何时该使用此节点。

选择器： [matTreeNodeDef]

属性
名称 描述
@Input('matTreeNode')

data: T

@Input(matTreeNodeDefWhen)

when: (index: number, nodeData: T) => boolean

如果该节点模板应该应用在所提供的节点数据和索引上，则该函数返回 true。如果未定义，那么当该函数对该数据返回 true 时，该节点将被认为是默认的节点模板。对于每一个节点，至少要传入一个 when 函数或者用 undefined 表示默认行为。

template: TemplateRef<any>

MatTree extends CdkTree

采用 Material Design 样式的 CdkTable 封装器。

选择器： mat-tree

Exported as: matTree
属性
名称 描述
@Input()

dataSource: DataSource<T> | Observable<T[]> | T[]

提供一个包含要渲染的最新数据数组的流。受树的视图流窗口影响（即当前屏幕上有哪些数据节点）。数据源可以是数据数组的可观察对象，也可以是要渲染的数据数组。
@Input()

trackBy: TrackByFunction<T>

跟踪函数，用于检查数据变化的差异。类似于 ngFor 的 trackBy 函数。 可以优化节点操作，方法是根据该函数处理后的数据来标识一个节点，以了解该节点是否应添加/删除/移动。接受带两个参数 index 和 item 的函数。
@Input()

这个流包含哪些节点正显示在屏幕上的最新信息。可以被数据源用作该提供哪些数据的线索。

MatTreeFlattener

树展平器，用于将普通类型的节点转换为具有子级和级别信息的节点。把 T 型嵌套节点转换为 F 型扁平节点。

例如，类型为 T 的输入数据是嵌套的，包含其子数据： SomeNode: { key: 'Fruits', children: [ NodeOne: { key: 'Apple', }, NodeTwo: { key: 'Pear', } ] } 扁平化器扁平化树后，结构将变为 SomeNode: { key: 'Fruits', expandable: true, level: 1 }, NodeOne: { key: 'Apple', expandable: false, level: 2 }, NodeTwo: { key: 'Pear', expandable: false, level: 2 } 输出扁平化类型为 F，并包含附加信息。

getLevel: (node: F) => number
