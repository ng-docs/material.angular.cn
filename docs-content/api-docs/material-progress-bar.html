Angular Material 进度条的 API 参考

import {MatProgressBarModule} from '@angular/material/progress-bar';

指令

MatProgressBar

选择器： mat-progress-bar

Exported as: matProgressBar
属性
名称 描述
@Input({ transform: numberAttribute })

bufferValue: number

进度条的缓冲值。默认为零。
@Input()

color: string

进度条的主题调色板颜色。
@Input()

mode: ProgressBarMode

进度条的模式吧。

此输入属性必须是以下值之一：determinate、indeterminate、buffer 和 query，默认为 'determinate'。会镜像到 mode 属性。
@Input({ transform: numberAttribute })

value: number

进度条的值。默认为零。镜像到 aria-valuenow。
@Output()

animationEnd: EventEmitter<ProgressAnimationEnd>

当主进度条的动画完成时会发出本事件。当禁用动画时，不会发出此事件，也不会为具有连续动画的模式（不定动画和查询动画）发出此事件。

接口

ProgressAnimationEnd

最后一个动画结束数据

属性
名称 描述

value: number

MatProgressBarDefaultOptions

可覆盖的默认 mat-progress-bar 选项。

属性
名称 描述

color: ThemePalette

进度条的默认颜色。

mode: ProgressBarMode

进度条的默认模式。

类型别名

ProgressBarMode 

type ProgressBarMode = 'determinate' | 'indeterminate' | 'buffer' | 'query';

常量

MAT_PROGRESS_BAR_DEFAULT_OPTIONS

用于覆盖 mat-progress-bar 的默认选项的注入令牌。

const MAT_PROGRESS_BAR_DEFAULT_OPTIONS: InjectionToken<MatProgressBarDefaultOptions>;
