Angular CDK 列表框 API 参考

import {CdkListboxModule} from '@angular/cdk/listbox';

指令

CdkOption

列表框中的可选选项。

选择器： [cdkOption]

Exported as: cdkOption
属性
名称 描述
@Input({ alias: 'cdkOptionDisabled', transform: booleanAttribute })

disabled: boolean

该选项是否已禁用。
@Input('tabindex')

enabledTabIndex:

此选项启用时的 tabindex。
@Input()

id:

选项的宿主元素的 ID。
@Input('cdkOptionTypeaheadLabel')

typeaheadLabel: string | null

用于在列表框预先输入（typeahead）期间定位此条目的文本。如果未指定，将使用条目的 textContent 。
@Input('cdkOption')

value: T

此选项的值。

destroyed: Subject<void>

当销毁此选项时发出。

element: HTMLElement

此选项的宿主元素

listbox: CdkListbox<T>

此选项所属的父列表框。
方法
deselect

如果已选定此选项，就取消选定它。
focus

聚焦此选项。
getLabel

获取 FocusableOption 接口所需的本元素的标签。
isActive

此选项是否处于活动状态。
isSelected

是否选定了此选项。
select

如果未选定此选项，就选定它。
toggle

切换此选项的选定状态。

CdkListbox

选择器： [cdkListbox]

Exported as: cdkListbox
属性
名称 描述
@Input('cdkListboxCompareWith')

compareWith: undefined | ((o1: T, o2: T) => boolean)

用于比较选项值的函数。
@Input({ alias: 'cdkListboxDisabled', transform: booleanAttribute })

disabled: boolean

此列表框是否被禁用。
@Input('tabindex')

enabledTabIndex:

启用此列表框时使用的 tabindex。
@Input()

id:

选项的宿主元素的 ID。
@Input({ alias: 'cdkListboxMultiple', transform: booleanAttribute })

multiple: boolean

此列表框是否允许选定多个选项。如果值从 true 切换为 false ，并且选定了多个选项，则取消选定所有选项。
@Input({ alias: 'cdkListboxNavigatesDisabledOptions', transform: booleanAttribute })

navigateDisabledOptions: boolean

键盘导航是否应跳过禁用的条目。
@Input({ alias: 'cdkListboxNavigationWrapDisabled', transform: booleanAttribute })

navigationWrapDisabled: boolean

当用户在最后一个条目上按下向下箭头或在第一个条目上按下向上箭头时，键盘导航是否应该回绕。
@Input('cdkListboxOrientation')

orientation: "vertical" | "horizontal"

此列表框的方向。只影响键盘交互，不影响视觉布局。
@Input({ alias: 'cdkListboxUseActiveDescendant', transform: booleanAttribute })

useActiveDescendant: boolean

此列表框要使用活动后代还是将焦点移到选项上。
@Input('cdkListboxValue')

value: readonly T[]

此列表框中选定的值，表示为选项值的数组。
@Output('cdkListboxValueChange')

valueChange: Subject<ListboxValueChangeEvent<T>>

当列表框中的选定值更改时发出。

changeDetectorRef: inject(ChangeDetectorRef)

此列表框的变更检测器。

destroyed: Subject<void>

当销毁此列表框时发出。

element: HTMLElement

此列表框的宿主元素。

listKeyManager: ActiveDescendantKeyManager<CdkOption<T>>

管理此列表框的键盘导航的键盘管理器。

ngZone: inject(NgZone)

Angular 区域（Zone）。

options: QueryList<CdkOption<T>>

此列表框中的子选项。

selectionModel: ListboxSelectionModel<T>

此列表框使用的选择模型。
方法
deselect

取消选择给定的选项。
参数

option

CdkOption<T>

要取消选定的选项
deselectValue

取消选定给定的值。
参数

value

T

要取消选定的值
focus

聚焦此列表框的宿主元素。
isActive

获取给定选项是否处于活动状态。
参数

option

CdkOption<T>

要获取活动状态的选项
返回值
boolean

isSelected

获取是否选定了给定的选项。
参数

option

CdkOption<T>

要获取选定状态的选项
isValueSelected

获取给定值是否已选定。
参数

value

T

要获取选定状态的值
select

选定给定的选项。
参数

option

CdkOption<T>

要选定的选项
selectValue

选定给定的值。
参数

value

T

要选定的值
setAllSelected

设置所有选项的选定状态。
参数

isSelected

boolean

要设置的新选定状态
toggle

切换给定选项的选定状态。
参数

option

CdkOption<T>

要切换的选项
toggleValue

切换给定值的选定状态。
参数

value

T

要切换的值
triggerOption

触发给定选项以响应用户交互。

  • 在单选模式下：选择此选项并取消选择任何其他选定的选项。
  • 在多选模式下：切换此选项的选定状态。
参数

option

CdkOption<T>

要触发的选项
triggerRange

触发给定范围的选项以响应用户交互。只应在多选模式下调用。
参数

trigger

CdkOption<T>

触发过的选项

from

number

要切换的选项的起始索引

to

number

要切换的选项的结束索引

on

boolean

是否开启选项范围

接口

ListboxValueChangeEvent

change 事件，当列表框的值改变时触发。

属性
名称 描述

listbox: CdkListbox<T>

对发出本事件的列表框的引用。

option: CdkOption<T> | null

对触发过的选项的引用。

value: readonly T[]

此列表框的新值。
