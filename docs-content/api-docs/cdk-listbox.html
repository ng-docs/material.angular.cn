Angular CDK listbox API 参考文档

import {CdkListboxModule} from '@angular/cdk/listbox';

指令

CdkOption

列表框中的可选选项。

A selectable option in a listbox.

选择器： [cdkOption]
导出为： cdkOption
属性
名称 描述
@Input('cdkOptionDisabled')

disabled: boolean

该选项是否已禁用。

Whether this option is disabled.
@Input('tabindex')

enabledTabIndex:

此选项启用时的 tabindex。

The tabindex of the option when it is enabled.
@Input()

id:

选项的宿主元素的 ID。

The id of the option's host element.
@Input('cdkOptionTypeaheadLabel')

typeaheadLabel: string

用于在列表框预先输入（typeahead）期间定位此条目的文本。如果未指定，将使用条目的 textContent 。

The text used to locate this item during listbox typeahead. If not specified, the textContent of the item will be used.
@Input('cdkOption')

value: T

此选项的值。

The value of this option.

destroyed: Subject<void>

当销毁此选项时发出。

Emits when the option is destroyed.

element: HTMLElement

此选项的宿主元素

The option's host element

listbox: CdkListbox<T>

此选项所属的父列表框。

The parent listbox this option belongs to.
方法
deselect

如果已选定此选项，就取消选定它。

Deselect this option if it is selected.
focus

聚焦此选项。

Focus this option.
getLabel

获取 FocusableOption 接口所需的本元素的标签。

Get the label for this element which is required by the FocusableOption interface.
isActive

此选项是否处于活动状态。

Whether this option is active.
isSelected

是否选定了此选项。

Whether this option is selected.
select

如果未选定此选项，就选定它。

Select this option if it is not selected.
toggle

切换此选项的选定状态。

Toggle the selected state of this option.

CdkListbox

选择器： [cdkListbox]
导出为： cdkListbox
属性
名称 描述
@Input('cdkListboxCompareWith')

compareWith: undefined | ((o1: T, o2: T) => boolean)

用于比较选项值的函数。

The function used to compare option values.
@Input('cdkListboxDisabled')

disabled: boolean

此列表框是否被禁用。

Whether the listbox is disabled.
@Input('tabindex')

enabledTabIndex:

启用此列表框时使用的 tabindex。

The tabindex to use when the listbox is enabled.
@Input()

id:

选项的宿主元素的 ID。

The id of the option's host element.
@Input('cdkListboxMultiple')

multiple: boolean

此列表框是否允许选定多个选项。如果值从 true 切换为 false ，并且选定了多个选项，则取消选定所有选项。

Whether the listbox allows multiple options to be selected. If the value switches from true to false, and more than one option is selected, all options are deselected.
@Input('cdkListboxNavigatesDisabledOptions')

navigateDisabledOptions: BooleanInput

键盘导航是否应跳过禁用的条目。

Whether keyboard navigation should skip over disabled items.
@Input('cdkListboxNavigationWrapDisabled')

navigationWrapDisabled: BooleanInput

当用户在最后一个条目上按下向下箭头或在第一个条目上按下向上箭头时，键盘导航是否应该回绕。

Whether the keyboard navigation should wrap when the user presses arrow down on the last item or arrow up on the first item.
@Input('cdkListboxOrientation')

orientation: "vertical" | "horizontal"

此列表框的方向。只影响键盘交互，不影响视觉布局。

The orientation of the listbox. Only affects keyboard interaction, not visual layout.
@Input('cdkListboxUseActiveDescendant')

useActiveDescendant: boolean

此列表框要使用活动后代还是将焦点移到选项上。

Whether the listbox will use active descendant or will move focus onto the options.
@Input('cdkListboxValue')

value: readonly T[]

此列表框中选定的值，表示为选项值的数组。

The value selected in the listbox, represented as an array of option values.
@Output('cdkListboxValueChange')

valueChange: Subject<ListboxValueChangeEvent<T>>

当列表框中的选定值更改时发出。

Emits when the selected value(s) in the listbox change.

changeDetectorRef: inject(ChangeDetectorRef)

此列表框的变更检测器。

The change detector for this listbox.

destroyed: Subject<void>

当销毁此列表框时发出。

Emits when the listbox is destroyed.

element: HTMLElement

此列表框的宿主元素。

The host element of the listbox.

listKeyManager: ActiveDescendantKeyManager<CdkOption<T>>

管理此列表框的键盘导航的键盘管理器。

The key manager that manages keyboard navigation for this listbox.

ngZone: inject(NgZone)

The Angular zone.

options: QueryList<CdkOption<T>>

此列表框中的子选项。

The child options in this listbox.

selectionModel: ListboxSelectionModel<T>

此列表框使用的选择模型。

The selection model used by the listbox.

triggerOption

触发给定选项以响应用户交互。

Triggers the given option in response to user interaction.

  • 在单选模式下：选择此选项并取消选择任何其他选定的选项。

    In single selection mode: selects the option and deselects any other selected option.

  • 在多选模式下：切换此选项的选定状态。

    In multi selection mode: toggles the selected state of the option.

Trigger the given range of options in response to user interaction. Should only be called in multi-selection mode.
