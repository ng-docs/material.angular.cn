Angular Material 消息提示的 API 参考

import {MatSnackBarModule} from '@angular/material/snack-bar';

服务

MatSnackBar

一个服务，用来派发 Material Design 快餐栏的消息。

属性
名称 描述

handsetCssClass: 'mat-mdc-snack-bar-handset'

用于手机模式的 CSS 类。

simpleSnackBarComponent: SimpleSnackBar

应作为消息提示的简单组件渲染的组件。

snackBarContainerComponent: MatSnackBarContainer

将提供的模板或组件附加的容器组件。
方法
dismiss

关闭目前可见的快餐栏。
open

带有信息和可选操作的快餐栏。
参数

message

string

要显示在快餐栏里的消息。

action

string = ''

快餐栏的动作标签。

配置 ?

MatSnackBarConfig<any>

快餐栏的其它配置选项。
返回值
MatSnackBarRef<TextOnlySnackBar>

openFromComponent

为内容创建并派发带有自定义组件的快餐栏，删除任何目前打开着的快餐栏。
参数

component

ComponentType<T>

要实例化的组件。

配置 ?

MatSnackBarConfig<D>

快餐栏的额外配置。
返回值
MatSnackBarRef<T>

openFromTemplate

使用该内容的自定义模板创建并派发一个快餐栏，删除任何目前打开着的快餐栏。
参数

template

TemplateRef<any>

要实例化的模板。

配置 ?

MatSnackBarConfig<any>

快餐栏的额外配置。
返回值
MatSnackBarRef<EmbeddedViewRef<any>>

指令

SimpleSnackBar

选择器： simple-snack-bar

Exported as: matSnackBar
属性
名称 描述

data: { message: string; action: string; }

hasAction: boolean

是否要显示动作按钮。

snackBarRef: MatSnackBarRef<SimpleSnackBar>

方法
action

执行快餐店里的动作。

MatSnackBarLabel

要应用于要在快餐栏中渲染的文本元素的指令。

选择器： [matSnackBarLabel]

MatSnackBarActions

要应用于包含快餐栏操作按钮的元素的指令。

选择器： [matSnackBarActions]

MatSnackBarAction

要应用于快餐栏的每个操作按钮的指令。

选择器： [matSnackBarAction]

类

MatSnackBarConfig

打开快餐栏时使用的配置。

属性
名称 描述

announcementMessage: string

将由 LiveAnnouncer 朗读的消息。当没有自定义组件或模板的情况下打开快餐栏时，公告信息会默认为指定的消息。

data: D | null

注入到子组件中的数据。

direction: Direction

快餐栏的文字布局方向。

duration: number

在自动关闭快餐栏之前要等待的毫秒数。

horizontalPosition: MatSnackBarHorizontalPosition

放置快餐栏的水平位置。

panelClass: string | string[]

要添加到快餐栏容器中的额外 CSS 类。

politeness: AriaLivePoliteness

MatAriaLiveAnnouncer 公告的文雅程度。

verticalPosition: MatSnackBarVerticalPosition

放置快餐栏的垂直位置。

viewContainerRef: ViewContainerRef

视图容器的引用，可以在依赖注入时作为快餐栏的父组件。注意：这不会影响快餐栏在 DOM 中的插入位置。

MatSnackBarRef

到快餐栏服务派发出的快餐栏的引用。

属性
名称 描述

instance: T

该组件的实例构成了快餐栏的内容。
方法
afterDismissed

获取一个当快餐栏结束时会得到通知的可观察对象。
返回值
Observable<MatSnackBarDismiss>

afterOpened

获取一个可观察对象，当快餐栏已经打开并出现时，会通知它。
返回值
Observable<void>

dismiss

关闭快餐栏。
dismissWithAction

标记某个快餐栏动作被点击过。
onAction

获取一个会在调用小吃条动作时得到通知的可观察对象。
返回值
Observable<void>

已弃用
closeWithAction

标记某个快餐栏动作被点击过。

接口

TextOnlySnackBar

简单快餐栏组件的接口，它包含一条消息和一个动作。

属性
名称 描述

action: () => void

data: { message: string; action: string; }

hasAction: boolean

snackBarRef: MatSnackBarRef<TextOnlySnackBar>

MatSnackBarDismiss

快餐栏被关闭时发生的事件。

属性
名称 描述

dismissedByAction: boolean

快餐栏是否被动作按钮关闭了。

类型别名

MatSnackBarHorizontalPosition

MatSnackBarConfig 中 horizontalPosition 的可选值。

type MatSnackBarHorizontalPosition = 'start' | 'center' | 'end' | 'left' | 'right';

MatSnackBarVerticalPosition

MatSnackBarConfig 中 verticalPosition 的可选值。

type MatSnackBarVerticalPosition = 'top' | 'bottom';

常量

MAT_SNACK_BAR_DEFAULT_OPTIONS

这个注入令牌可以用来指定快餐栏的默认配置。

const MAT_SNACK_BAR_DEFAULT_OPTIONS: InjectionToken<MatSnackBarConfig<any>>;

MAT_SNACK_BAR_DATA

这个注入令牌可以用来访问传入快餐栏的数据。

const MAT_SNACK_BAR_DATA: InjectionToken<any>;
